# YouStart AI

**Protótipo de IA Generativa para Creators da YouShop**  
Desafio de Inovação com IA — Programa de Estágio YouShop 2026

---

## 🎯 Sobre o Projeto

YouStart AI é um assistente inteligente que ajuda creators e afiliados da YouShop a:

- Descobrir o melhor produto para promover
- Receber um plano de ação personalizado de 7 dias
- Gerar copies persuasivas prontas para usar
- Superar seus principais obstáculos com estratégias de conteúdo

Desenvolvido como parte do processo seletivo do Programa de Estágio YouShop 2026.

---

## ✨ Funcionalidades

- Interface moderna e intuitiva
- Fluxo guiado com 3 perguntas
- **Integração real com Google Gemini 2.5 Flash** (IA Generativa)
- Geração dinâmica de plano de ação, copy e recomendações
- Design responsivo
- Totalmente funcional como Single Page Application

---

## 🛠️ Tecnologias Utilizadas

- HTML5, CSS3 e Vanilla JavaScript
- Google Gemini API (via variável de ambiente)
- Vercel (deploy)
- Modularização com ES Modules
